# Ejemplos sintéticos de campo

Estos archivos son **datos sintéticos para prueba técnica**. No son observaciones reales y no deben citarse como evidencia empírica.

Reglas:

- No cambian `pending_no_capture`.
- No se publican en capítulos como datos reales.
- Sirven para revisar columnas, formatos y validadores.
- El script `ingest_fieldwork.py` excluye la carpeta `examples/` del descubrimiento de sesiones reales.

Para probar manualmente un ejemplo, copiar la carpeta `synthetic_2026_04_25/` fuera de `examples/` con un nombre de sesión temporal, ejecutar ingesta y borrar la copia después. No hacer esto en una corrida de tesis final.
