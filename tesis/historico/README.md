# Histórico de tesis

Esta carpeta registra cambios críticos en la tesis a lo largo del tiempo. No reemplaza a Git: funciona como bitácora académica legible para reconstruir decisiones de escritura, cambios de alcance, autocorrecciones y advertencias metodológicas.

## Regla de archivo

- Usar formato `AAAA-MM-DD-tema.md`.
- Registrar qué cambió, por qué cambió y qué riesgos quedan.
- No mover aquí documentos activos de trabajo de campo ni datos.
- Mantener `tesis/pendientes/` para tareas vivas y `tesis/historico/` para auditorías cerradas.
